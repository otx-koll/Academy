#### 3. Scanner를 이용하여 정수를 입력받고 다음과 같이 \*를 출력하는 프로그램을 작성하여라.

```
import java.util.Scanner;

public class ex03_3 {

	public static void main(String[] args) {
		Scanner s = new Scanner(System.in);
		
		System.out.print("정수를 입력하시오>>");
		int n = s.nextInt();
		
		for(int i=n; i>0; i--) {
			for(int j=0; j<i; j++)
				System.out.print("*");
			System.out.println();
		}
	}
}
```
출력 결과
```
정수를 입력하시오>>5
*****
****
***
**
*
```
