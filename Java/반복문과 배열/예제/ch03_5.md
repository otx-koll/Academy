#### 5. for문과 continue문을 사용하여 1부터 100까지 짝수의 합을 구해보자

```
public class ch03_5 {

	public static void main(String[] args) {
		int sum = 0;
		for(int i=1; i<=100; i++) {
			if(i%2 == 1)
				continue;
			else
				sum += i;
		}
		System.out.println("1부터 100까지 짝수의 합은 "+sum);
	}
}
```
출력 결과
```
1부터 100까지 짝수의 합은 2550
```
